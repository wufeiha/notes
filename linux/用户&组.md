用户操作
==
**1、添加用户**
```shell
useradd -g groupname username
```
**2、修改用户**
```shell
usermod 
```

**3、删除用户**
```shell
userdel username
```

**4、查看所有用户及相关信息**
```shell
cat /etc/passwd
```
**5、/etc/passwd 文件各参数介绍**

**6、将用户改为不可登陆用户**

```shell
usermod -s /sbin/nologin username
```

**7、不可登陆用户如何执行命令**
方式一：推荐

```shell
su -s /bin/bash -c "command line" username
```
方式二：不推荐    以下命令会fork一个子进程
```shell
sudo -u username command line 
```
